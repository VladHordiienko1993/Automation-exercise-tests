BUG_002: Search returns irrelevant products

Severity: High
Priority: High
Status: Open
Found in: TC_007

Description:
Product search returns items that don't contain the search query. When searching for "dress", results include products like "sleeves top and short - blue & pink" that don't contain the word "dress".

Steps to Reproduce:
1. Navigate to /products
2. Enter "dress" in search input
3. Click search button
4. Check product names in results

Expected Result:
All product names should contain "dress"

Actual Result:
Some products don't contain search term (e.g., "sleeves top and short - blue & pink")

Impact:
Search functionality unreliable - users get irrelevant results
